<h1 align="center">🫸 Límites y Alcances</h1>

El proyecto `Absolute Cinema`, en su primera versión,
se enfocará en resolver los problemas más fundamentales
del cine y establecer una base sólida para futuras
mejoras y expansiones.

## Alcance 🎯

En esta primera fase, el sistema cubrirá únicamente
la operación básica de un cine pequeño:

- Gestión de cartelera (películas activas e inactivas)
- Gestión de salas y programación de funciones
- Publicación de horarios para consulta de clientes
- Venta básica de boletos en línea
- Registro de ventas en taquilla
- Reporte diario básico de ventas por función

## Límites 🚧

En esta primera fase, no se ven contemplados los siguientes aspectos:

- Gestión de promociones y descuentos
- Gestión de inventario de alimentos y bebidas
- Gestión de recursos humanos
- Programas de fidelización, membresías y cupones

## Soluciones ✅

El proyecto se centrará en desarrollar una plataforma mínima viable
que permita al cine cubrir sus necesidades operativas básicas y
ofrecer una experiencia mínima a los clientes, con funcionalidades como:

- **Panel administrativo** web para operación interna
- **Sitio cliente** para cartelera, horarios y compra básica
- **Base de datos central** para películas, funciones y ventas
- **Reportería** inicial orientada al control operativo diario

## Plataforma de desarrollo

Para esta primera fase, se optará por una plataforma de
desarrollo web tradicional con tecnologías como:

- **Python 3.14** como lenguaje de programación principal
- **Reflex 0.9.0** como framework de desarrollo web de Python
- **PostgreSQL 17** como sistema de gestión de bases de datos
- **Podman 4.9.3** para contenerización y despliegue local
- **Ubuntu 24.04** como sistema operativo de desarrollo y despliegue
- **Git** para control de versiones y colaboración
- **GitHub** para alojamiento del código y documentación
