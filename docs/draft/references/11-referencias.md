# 11. Referencias Bibliográficas

## Propósito de esta sección

La sección de referencias lista todas las fuentes consultadas para el desarrollo del proyecto, siguiendo un formato estandarizado.

## Formato de referencias

### Según el estilo APA 7ma edición

**Libros**:

```
Apellido, A. A. (Año). Título del libro. Editorial.
```

**Artículos**:

```
Apellido, A. A. (Año). Título del artículo. Nombre de la Revista, volumen(número), páginas.
```

**Documentos web**:

```
Apellido, A. A. (Año, Día Mes). Título de la página. Nombre del Sitio. URL
```

## Errores comunes a evitar

1. **No inventar referencias**: Solo incluye fuentes reales
2. **No mezclar formatos**: Usa un solo estilo en toda la sección
3. **No olvidar citas del documento**: Todo lo citado debe estar referenciado
4. **No usar fuentes no confiables**: Prioriza documentación oficial y publicaciones académicas
