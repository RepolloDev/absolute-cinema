<h1 align="center">🧐 Definición</h1>

El objetivo del proyecto `Absolute Cinema` ✋🧐🤚 es
ser una plataforma de gestión y experiencia para cualquier
cine, pero con un enfoque especial en cines pequeños y
medianos, como el "_Cine Monje Campero_", que es el caso
de estudio para este proyecto.

## Administración

La plataforma se encargará de gestionar la parte administrativa
de un cine, un sistema de información que ofrece:

- Aplicación web como medio de interacción 🌐
- Gestión de recursos humanos (empleados, roles, etc.) 🫂
- Gestión de inventario (películas, alimentos, etc.) 🍿
- Finanzas (ingresos, gastos, etc.) 💰
- Reportes y análisis (ventas, tendencias, etc.) 📊

## Experiencia del cliente

Además de la parte administrativa, la plataforma también se enfocará
en mejorar la experiencia del cliente, ofreciendo:

- Sitio web personalizable para cada cine 🌐
- Información actualizada sobre la cartelera de películas 🎬
- Información sobre instalaciones y servicios del cine 🎟️
- Información sobre alimentos y bebidas disponibles 🍽️
- Compras de boletos en línea 🎫
