<h1 align="center">🧐 Definición</h1>

El objetivo del proyecto `Absolute Cinema` ✋🧐🤚 es
desarrollar una plataforma básica de gestión y experiencia
para cines pequeños y medianos, tomando como referencia
el "_Cine Monje Campero_" como caso de estudio inicial.

En esta primera instancia, el proyecto prioriza resolver
la operación mínima diaria de un cine con una sola sucursal,
dejando las funcionalidades avanzadas para fases posteriores.

## Administración 💼

La plataforma se encargará de gestionar la parte administrativa
esencial del cine mediante una aplicación web, ofreciendo:

- Gestión de películas en cartelera (alta, edición y estado) 🎬
- Gestión de salas y horarios de funciones 🏛️
- Registro básico de ventas de boletos (taquilla/online) 🎫
- Reporte operativo básico de ventas por función y por día 📊

## Experiencia del cliente 👥

Además de la parte administrativa, la plataforma se enfocará
en cubrir la experiencia mínima del cliente, ofreciendo:

- Sitio web informativo con cartelera y horarios actualizados 🌐
- Detalle básico de cada película (sinopsis, género y duración) 📝
- Visualización de disponibilidad general por función 🎟️
- Compra de boletos en línea para funciones disponibles 🎫
